== Description ==

WordPress Responsive Vertical Icon Menu plugin is an nice and handy plugin for showing your menus in widget. It's very simple to use. There are a lot of style and font settings that you will like.

### View our plugin demo page:

*   [Demo Page](http://demo.wpdevart.com)

You can check some of our plugin features below.

### Features of Free version

*   **Easy install and use**
*   **Works without problem with all WordPress versions**
*   **Fully Responsive**
*   **Ability to add Icons**
*   **Ability to choose Submenu opening type and set the duration**
*   **Ability to choose icon function when clicking on it**
*   **User friendly back-end**
*   **Tested on popular WordPress themes**

You can upgrade the free version to [Vertical menu Pro](https://wpdevart.com/wordpress-vertical-menu-plugin) to add some great features.

### The features of Pro version

*   **Ability to change Main, Sub and Active menus styles**
*   **Ability to select Font family**
*   **Ability to set border styles**
*   **Ability to set Background color**

### If you have any type of question occuring our plugin contact us at support@wpdevart.com.

== Installation ==

### First of all download the plugin ZIP file.  

1. Log in to your WordPress admin panel.   
2. Go to your Plugins page, then click Add New > Upload Plugin .    
3. Click "Choose file" then select the plugin zip file. 
4. After that upload and activate our plugin.
